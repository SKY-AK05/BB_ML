Rules & Tips

Annotation Rules

RULE 1 80% Confidence Required
Only annotate if you are at least 80% sure it is a license plate.
RULE 2 50% Plate Must Be Visible
Annotate only if at least half the license plate is visible in the image.
RULE 3 All Edges Must Be Visible
All 4 edges of the license plate must be clearly visible before annotating.
RULE 4 Tight Box - No Gaps
Draw the box as tight as possible around the plate edges - no extra space.
